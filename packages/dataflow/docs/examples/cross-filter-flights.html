Cross-Filter Flights 10M

This example demonstrates interactive cross-filtering of 10 million flight records using Mosaic vgplot and DuckDB-WASM.

Key features shown:

  • Loading a remote Parquet file with 10M rows via DuckDB-WASM
  • Linked histograms with brush-based cross-filtering
  • Selections in one chart automatically filter the others

Based on the Observable notebook by UW Data.
